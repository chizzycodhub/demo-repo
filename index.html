A practise on how to use Git Command

GIT COMMAND

The following are some Git commands:

  1. clone:Brings a project hosted somewhere like Github into a folder on your local machine
  2. add:Track your files and changes on Git
  3. commit:Save files in Git
  4. push:Upload Git commits from local machine to a remote repo, like Github
  5. pull:Download changes from remote repo, like Github to your local machine. It is the opposite of push.
  6. ls -la:List everything in the the directory including the hidden files


GIT BRANCHING

The follwing as some git branching command

  • git branch {BRANCHNAME}:To create a branch
  • git branchTo check the branches available
  • git branch -d {BRANCHNAME}:To delete a branch
  • git branch -M {BRANCHNAME}:To rename a branch
  • git checkout {BRANCHNAME}:To switch between branch
